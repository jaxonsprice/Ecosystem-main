Node.js:

üéà Usage

This simulation uses Euler physics integration. The velocity property of the fish is influenced by the method addForce(), which takes a p5.Vector object.

// create a p5 Vector class. 
force = createVector(-1, 0)

// Moves the fish horizontally.
fish.addForce(force)
